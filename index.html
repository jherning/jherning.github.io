This website is primarily for my open source coding projects. I'm a “tinkerer” when it comes to coding, not a professional by any means.



Larger Projects

link68: Computer <-> calculator linking program for the Texas Instruments family of 68k-based calculators (TI-89 / 92 / 92+ / V200). Python. Download latest release. Source on Github.

Horizontal Precalculus: A non-standard precalculus resource ordered by difficulty instead of major topic. Consists of short notes and an accompanying problem set for each lecture. Download latest PDF. LaTeX on Github.

Professor Herning: A YouTube channel dedicated mostly to slide rules. I started making the videos when I was learning how to use them myself.


Small Stuff

Dr. Joe's Short Trigonometry Guide: A one-page (front/back) cheat-sheet for precalc/calc students.

Two-Tail t-Distribution Calculator: A nomogram for calculating Student's distribution. Cut vertically along the left side and move the probability scale to your desired DOF.
